## 📦 Projetos

- 💰 **Folha de Pagamento**  
  Aplicação em C# para cálculo de salário líquido com base em regras de INSS e IRRF. Possui validação de entrada, processamento automático dos descontos e exibição de holerite no console.

- ⚔️ **Item RPG**  
  Sistema de gerenciamento de itens com controle de estoque, vendas e reabastecimento, utilizando encapsulamento e validações.

- 🎵 **Portal de Música**  
  Aplicação com menu interativo para análise de um catálogo musical em memória, utilizando LINQ para filtros, ranking e métricas.

- 🚗 **Gestão de Frota**  
  Sistema que simula operações com veículos utilizando herança, polimorfismo e interfaces, incluindo consumo de combustível, pedágio e manutenção.

- 🎮 **Space Invaders (Console)**  
  Jogo em C# inspirado no clássico Space Invaders, com controle em tempo real, movimentação de entidades, detecção de colisões e sistema de pontuação.

- 🧩 **Doninha Encantada (TCB)**  
  Sistema completo de loja RPG com cadastro de itens, controle de estoque, vendas, relatórios e fechamento de caixa, utilizando arquitetura orientada a objetos.

- 🐉 **Catálogo de Monstros**  
  Sistema de cadastro e gerenciamento de entidades com foco em manipulação de dados e persistência.

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO): herança, encapsulamento, polimorfismo e abstração  
- Manipulação de coleções com `List<T>`  
- Consultas e manipulação de dados com LINQ  
- Estruturação de sistemas com separação de responsabilidades  
- Operações CRUD (criação, leitura, atualização e exclusão)  
- Validação de dados e tratamento de exceções  
- Persistência de dados (JSON e arquivos de texto)  
- Lógica de jogos em tempo real (game loop e colisões)
