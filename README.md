## 🧠 Conceitos aplicados

Durante o desenvolvimento dos projetos, foram aplicados os seguintes conceitos:

- Programação Orientada a Objetos (POO): herança, encapsulamento e polimorfismo  
- Manipulação de coleções com `List<T>`  
- Consultas e manipulação de dados com LINQ  
- Persistência de dados utilizando JSON  
- Registro de logs em arquivos de texto (.txt)  
- Operações CRUD (criação, leitura, atualização e exclusão de objetos)
